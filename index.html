🚀 How to run the application

This repository contains a full-stack application. To run the frontend locally:

  1. Clone this repository
  2. Navigate to the frontend directory: cd frontend
  3. Install dependencies: bun install
  4. Start the development server: bun run dev

Note: This page is just a static placeholder for GitHub Pages. The actual application uses Nuxt/Vue and runs via a node/bun server.
